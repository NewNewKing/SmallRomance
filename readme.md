# Merry Christmas (Unfinished)

### 目录结构

* dist  --打包后的文件
  - index.html 打包后的index文件 直接打开即可看到内容
  - js 打包后的js文件
* node_modules 
* src
  - config --配置文件主要为需要加载的img列表
  - img  --img文件
  - js --js文件
* index.html --模版主页面
* webpack.config.js

### 快速预览
访问[这个网站](https://newnewking.github.io/MerryChristmas/)

### 开发者调试

```
npm install
```

```
npm run dev
```
