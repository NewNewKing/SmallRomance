或者访问[github.io网站](https://newnewking.github.io/MerryChristmas/)


### 开发者调试

```
npm install
```

```
npm run dev
```

打包
```
npm run build
```

### 改动配置
大部分的参数配置都在`src/config/global.js`里面

只需改动参数就可开箱即用。

参数说明相见global.js注释
