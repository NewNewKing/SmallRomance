# Merry Christmas

### 快速预览
访问[这个网站](https://newnewking.github.io/MerryChristmas/)

### 名不副实
这个项目和圣诞节的关联不大，是作者本来想在圣诞节之前完成

结果太监了。

### 开发者调试

```
npm install
```

```
npm run dev
```

打包
```
npm run build
```

### 改动配置
大部分的参数配置都在`src/config/global.js`里面

只需改动参数就可开箱即用。

参数说明相见global.js注释
